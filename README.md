# Aemitez_Dev Infographic

This project is an interactive infographic showcasing Aemitez_Dev's freelance services, built with HTML, CSS (Tailwind), and JavaScript (Chart.js).

## Overview

The infographic visualizes key aspects of Aemitez_Dev's offerings:

- **Tech Stack**: A radar chart displaying full-stack capabilities.
- **Service Matrix**: A bubble chart illustrating time vs. cost tradeoffs.
- **Target Audience**: A doughnut chart showing client demographics.
- **Quality Metrics**: A polar area chart highlighting benefits like SEO and speed.
- **Workflow**: Step-by-step process visualization.
- **Call to Action**: Link to Fastwork profile.

## Features

- Responsive design using Tailwind CSS.
- Interactive charts powered by Chart.js.
- Custom styling with a Cyber Tech color palette.
- Thai language support.

## Usage

Open `index.html` in a web browser to view the infographic.

## Technologies Used

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Chart.js)
- Google Fonts (Kanit, Roboto)
